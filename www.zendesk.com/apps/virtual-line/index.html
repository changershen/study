Virtual Line

Virtual Line is one of the easiest tools to create cloud telephony solutions

(0)
Install

Free

Available only to agents with “Talk - Partner Edition” seats (see zendesk.com/talk/pricing ).

Virtual Line is one of the easiest and most powerful tools to create telephony solutions over the Internet, wherever you are. By adopting the block design, we make it easier for everyone to set up telephony systems without the need for IT support. Now you can sound professional by greeting customers with personalised messages, and manage service requests efficiently over an interactive voice response (IVR) menu with our text-to-speech software - all at a fraction of the cost of traditional phone systems.

Features:

  • Ring Queue
  • Call Forwarding
  • Multi-layered IVR Menu
  • Greeting Message Customisation
  • Language Selection Menu
  • Voicemail Service
  • Office Hours & Work Day Customisation

You can sign up for Virtual Line either by signing up or logging into www.hoiio.com. After getting to the web portal you may continue to subscribe to the virtual line application under Hoiio's app center. In addition, you may pick numbers from different countries (Singapore, Hong Kong, Malaysia, and so forth). And after that you can set up the virtual line by using simple blocks and layers and afterward you are prepared to course to various departments and numbers.
